# 使用技術
- js module 
1.  把主程式拆開，將每一個獨立的完整功能拆開成模組，使主程式乾淨與好管理
- webpack 
1.  將所有 js 檔案壓縮成一個 bundle.js，減少 request
2.  管理使用的套件
- typed.js 套件
1.  使文字顯示有人工打字逐字出現的效果
